Project Overview

This project focused on designing and evaluating a structural beam cross section for a communication satellite under specified loading and stiffness constraints. I compared candidate geometries, computed section properties, and validated performance using analysis and (optional) FEA.

Role: Student Engineer Tools: SolidWorks / Simulation, MATLAB, Excel Skills: Beam theory, stress, deflection, FoS Deliverables: Plots, calcs, final cross-section
